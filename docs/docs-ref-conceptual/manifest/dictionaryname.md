# <a name="dictionaryname-element"></a>Elemento DictionaryName

Especifica o nome deste dicionário.

**Tipo de suplemento:** Painel de tarefas

## <a name="syntax"></a>Sintaxe

```XML
<DictionaryName DefaultValue="string" />
```
